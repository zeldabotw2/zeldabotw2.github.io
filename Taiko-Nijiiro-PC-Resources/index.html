《太鼓達人》PC 資源整理（隨時更新）

發表於2024-03-03|更新於2025-02-10|太鼓達人
|閱讀量:

安這裡台輟，鑒於網路上太鼓的資源相當零散，且多數也已過時失效。故在這邊匯整一下，給大家一點方向。

若有任何變動而這邊來不及更新，就再麻煩各位下面留言告知了。我看到就會補上。

目前最佳解：TNDE

載點：https://ux.getuploader.com/norisuke/download/5

注意這個包並不是打開即用，需要再進 Readme 看一下，裡面有 theme 的下載點，下載後放入 system 裡面才能正常開啟。

需要 Locale Emulator 嗎？

其實根本不用！系統裡面設定一下日文環境就好。

設定方法參考這裡：https://guide.tjadataba.se/

大江戸勘亭流字體

這裡：https://guide.tjadataba.se/

目前最方便解：網頁版太鼓

眾所周知，網頁版太鼓前些時日紅紅火火，所以果不其然被四處追究下架。

但因為當初這個專案就是開源的，早已經被 fork 出去一大堆了，厲害了我的時代！

不過雖然源碼有很多分靈體，但每當架起來給大家玩，就很容易被官方找到，所以可能都不會太長壽。

這裡列出一些目前還活著的：

https://cjdgrevival.com/

https://obscuretaiko.glitch.me/

https://taiko.ani-nya.com/

如果以上有失效，或是有新的分靈體，請下面留言告知，我再更新。

OpenTaiko

TJAPlayer 系列唯一目前還有在公開更新迭代的專案：https://github.com/0auBSQ/OpenTaiko

不過要注意的是，似乎是沒有人在製作更新後的太鼓皮膚（主題）。

TJAPlayer3

最後就停在這了，沒有繼續迭代：https://github.com/twopointzero/TJAPlayer3

曲包

建議大家到「虹色+α 雑談&配布鯖」這個 Discord 群，這裡有維護 up-to-date 且完整的虹色街機曲包。直接搜尋「虹色+α 雑談&配布鯖」應該就會有進群入口，但如果找不到，可以私訊台輟，我拉你進。

Meowgister 個人整理的官方曲包：https://drive.google.com/drive/folders/1_HtQzaHeiCmSYTE3NYI44SGG81ldsyHR

Harry 個人製作的官方曲：https://harry-haifu.f5.si/ （雖然比較零星，但會放較新的曲目）

網友個人製作的 2023 道館課題包：https://drive.google.com/drive/folders/1g-6bXzT1hsUJnDyomW22TDTcz0QHruDR

主題（皮膚、skin）

あきら Ch.'s OpenTaiko Nijiiro Skin：https://drive.google.com/file/d/1G6nXCrQ7W_2nh9NpkntlSpEgeGWxBanE/view/ （但套用到新版本的 OpenTaiko 似乎都會報錯，不太確定原因）

其他酷東西

真正意義上的街機模擬器：https://teknoparrot.com/

TaikoArcadeLoader：https://github.com/BroGamer4256/TaikoArcadeLoader

TaikoLocalServer：https://github.com/asesidaa/TaikoLocalServer
