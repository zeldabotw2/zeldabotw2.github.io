安這裡台輟，有鑒於手把玩家跟鍵鼠玩家常常存在溝通上的困難，這邊特別列表，供大家參考。

武器操作

PS 鍵鼠 手把
舉起武器/
一般攻擊
特殊攻擊
同時按下動作 ➕ 或
➕		 ➕
防禦/特殊武器動作

投射器操作

手把 鍵鼠
投射器瞄準 右鍵長按
投射器發射 滾輪中鍵
發射鉤鎖（投射器瞄準時） F
選擇導蟲通知（投射器瞄準時） PgUp / PgDn / Q
選擇導蟲目標（投射器瞄準時） Tab

獵人操作

手把 鍵鼠
移動 左蘑菇頭 WASD
衝刺 左 Shift
蹲下/迴避 空白鍵
調查/對話/採集/剝取 F
使用道具 E
打開開始選單 Esc 或 H

鷺鷹龍操作

手把 鍵鼠
呼叫鷺鷹龍（自動導航模式） Tab
呼叫鷺鷹龍（手動移動模式） Q
切換主副武器 X
加速（按一次）/ 全速奔跑（長按） 左 Shift
減速（按一次）/急停（長按） Z
從鷺鷹龍背上下來 空白鍵
確認鷺鷹龍包 N

交流操作

手把 鍵鼠
聊天發言 Enter
一按通 T
貼圖 F9
動作 F10
姿勢 F11
成員清單 L
