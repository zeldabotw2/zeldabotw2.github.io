《黑帝斯》如何適配 21:9 寬螢幕？

發表於2024-11-07|更新於2024-11-07|黑帝斯
|閱讀量:

黑帝斯不適配嗎？

黑帝斯有原生適配超寬螢幕，台輟這邊拿 3440x1440 的螢幕進去，該分辨率默認是可供選擇的。

但事實上，開發團隊的做法是……

由圖可見，兩側超過 16:9 比例以外的畫面，都用美術圖蓋住了。

我不要啊！明明技術上地圖是有加載的狀態，那我都要看到！我也想要呈現這樣的畫面：

今天來教大家怎麼弄，走起！

Hephaistos

這是來自工程師玩家的開源項目，可以讓黑帝斯完美適配各種螢幕比例。無論是 21:9 或是 32:9 的 ultrawide 螢幕，或是 48:9 的 multi-monitor，甚至是 16:10 的 Steam Deck，都可以無黑邊無美術遮擋運行。

以 windows 平台為例，我們在 Hephaistos 的 github 頁面中找到 release，下載最新版本的執行檔，解壓縮後可以得到一個 hephaistos.exe。

把這個檔案放在黑帝斯資料夾裡面，如圖：

接著雙擊執行，按照命令行的提示互動操作即可，十分親民！
